阿里云镜像仓库可以代理k8s.gcr.io镜像

===>docker pull k8s.gcr.io/pause:3.2 docker pull registry.cn-hangzhou.aliyuncs.com/google_containers/pause:3.2 docker pull registry.cn-hangzhou.aliyuncs.com/dockerhub/alpine

dockerhub镜像仓库可以代理k8s.gcr.io镜像

#示例 docker pull k8s.gcr.io/pause:3.1 docker pull k8s.gcr.io/kube-apiserver:v1.17.3

#改为 docker pull googlecontainersmirrors/pause:3.1 docker pull googlecontainersmirrors/kube-apiserver:v1.17.3

七牛云镜像仓库可以代理quay.io镜像

#示例 docker pull quay.io/kubernetes-ingress-controller/nginx-ingress-controller:0.30.0

#改为 docker pull quay-mirror.qiniu.com/kubernetes-ingress-controller/nginx-ingress-controller:0.30.0
