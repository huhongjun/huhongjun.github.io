  • 基于README文件标注文件与目录，通过hover tooltip可查看
  • mkdocs扩展 基于文件首行的注释半自动生成json目录页 预览
  • 工作环境上下文相关的命令备忘录
  • 选中文本的自定义帮助中心查询
  • markdown表格与csv转换
  • AWS EC2 explorer
  • 为Ubuntu或某个常用软件如nginx奖励虚拟workspce 虚拟形式文件树展示 某个软件的全部配置操作文件的链接 工作目录和配置文件 ==>并且能支持多种版本的os+nginx组合==>可以ugc=>可以标记注释,把弹幕发扬=>标记注释学习=>知识上下文减少学习的需要,搜索的需要
