ESXi

安装

- 硬件：X220i，2CPU+8G+SSD(250G)
- 软件：ESXi 6.7 U2

配置

IP

静态IP 192.168.1.153,255.255.255.0

NFS数据存储

ds215j共享文件夹配置nfs访问

远程syslog服务器

esxcli esxcli system syslog config get esxcli system syslog config set --loghost='tcp://10.11.12.13:514' esxcli system syslog reload nc -z 192.168.1.122 514 esxcli network firewall ruleset set --ruleset-id=syslog --enabled=true esxcli network firewall refresh

Windows 10环境 ssh访问

- MS Windows 10安装openssh客户端
- 免密访问
~~~
cmd
ssh root@192.168.1.153
ssh-copy-id -i C:\Users\huhon\.ssh\id_rsa.pub root@192.168.1.153
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
~~~

ESXi操作指南

数据存储

ESXi日常维护

cd /vmfs/volumes/5de48648-dc383581-7e6c-f0def17d43c7/dsm esxcli storage -h

安装usb网卡驱动

cd /vmfs/volumes/de909bd1-1ffce6c4/VMWare/vssphere esxcli system maintenanceMode set -e true esxcli software vib install -d ESXi670-VMKUSB-NIC-FLING-33242987-offline_bundle-15615590.zip

Step 4 - Plug-in the USB NIC and reboot for the change to go into effect. Once the host has rebooted, ESXi should automatically pickup and claim the USB NIC (e.g. vusb0)
