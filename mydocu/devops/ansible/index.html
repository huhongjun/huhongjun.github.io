Ansible入门

Ubuntu 18.04 安装Ansible

sudo apt update
sudo apt install software-properties-common
sudo apt-add-repository ppa:ansible/ansible
sudo apt update
sudo apt install ansible
ansible --version

配置

远程被管理服务器aws ec2 sudo nano ./.ssh/authorized_keys // 加入公钥id_rsa.pub，一个一行

本地192.168.1.114 .ssh存放私钥

ssh ubuntu@18.163.140.251
sudo apt update
sudo apt install python

练习

sudo nano /etc/ansible/hosts

[servers]
awsss ansible_ssh_host=18.163.140.251
ws1804 ansible_ssh_host=192.168.1.133

sudo mkdir /etc/ansible/group_vars sudo nano /etc/ansible/group_vars/servers sudo nano /etc/ansible/group_vars/all

sudo mkdir /etc/ansible/host_vars

sudo nano /etc/ansible/host_vars/awsss ~~~ yaml --- ansible_ssh_user: ubuntu ~~~

sudo nano /etc/ansible/host_vars/ws1804 ~~~ yaml --- ansible_ssh_user: huhongjun ~~~

ansible -m ping servers ansible -m shell -a 'free -m' awsss ansible awsss -m command -a 'w'

远程安装ssh秘钥

ssh-copy-id -i ~/.ssh/id_rsa.pub huhongjun@192.168.1.133 ssh 'huhongjun@192.168.1.133'

ssh-copy-id -i ~/.ssh/id_rsa.pub admin@192.168.1.172
