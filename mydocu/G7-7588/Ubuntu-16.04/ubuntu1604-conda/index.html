ubuntu 16.04 conda环境安装

安装

安装anaconda 3

下载Anaconda3-2019.10-Linux-x86_64.sh sh Anaconda3-2019.10-Linux-x86_64.sh 根据提示完成安装

配置镜像源-conda

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

配置镜像源-pip

nano ~/.pip/pip.conf

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple

镜像源

  • a.豆瓣 https://pypi.douban.com/simple
  • b.阿里云： http://mirrors.aliyun.com/pypi/simple/
  • c.清华： https://pypi.tuna.tsinghua.edu.cn/simple

使用

配置conda

conda config --set auto_activate_base false

  • 配置文件

vi .bashrc

# >>> conda initialize >>>
# !! Contents within this block are managed by 'conda init' !!
__conda_setup="$('/home/huhongjun/anaconda3/bin/conda' 'shell.bash' 'hook' 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__conda_setup"
else
    if [ -f "/home/huhongjun/anaconda3/etc/profile.d/conda.sh" ]; then
        . "/home/huhongjun/anaconda3/etc/profile.d/conda.sh"
    else
        export PATH="/home/huhongjun/anaconda3/bin:$PATH"
    fi
fi
unset __conda_setup
# <<< conda initialize <<<
