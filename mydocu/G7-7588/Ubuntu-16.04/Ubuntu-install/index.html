Ubuntu Desktop 16.04 LTS on G7-7588

安装ubuntu

  1. UltraISO制作Ubuntu启动U盘；

  2. 启动机器，出现Dell标志时按F12，进入关闭安全启动，保存BIOS后重启F12启动选项菜单，选择U盘启动；

  3. 选择安装ubuntu（nomodeset - 显卡驱动问题）； 选择下载更新和第三方驱动自动安装； 分区设置： 选择grub引导安装在第二个硬盘; 分区（/boot, swap 8G，/ 54G）

  4. 操作系统关系

    选择其它

  5. 安装重启
