DEll G7-7588 Ubuntu 16.04

简介

G7系列有一款机器通过了Canical的Ubuntu16.04认证。

使用指南

常用指令

uname -a
kill -9 PID
dpkg --get-selections | grep firefox
sudo apt-get purge firefox   firefox-globalmenu firefox-locale-en   firefox-locale-zh-hans
source /etc/profile
.bashrc中PS1=行中的小写w改为大写的W，显示目录最后一段
sudo ntfsfix /dev/sda3
sudo mount -t nfs 192.168.1.122:/volume2/NFS-ds3617xs /home/huhongjun/mnt-ds215j-nfs-ds3617xs

df -h 查看空间使用情况
sudo du -sh

Ubuntu分区调整

插入Ubuntu启动U盘；
F12选择U盘=》选择使用不安装Ubuntu（需要nomodeset）；
sudo Gparted；
图形化界面分区操作；
重启；

加载NTFS分区问题

sudo ntfsfix /dev/sda3

挂载新硬盘并迁移home目录

  • 添加硬盘

sudo fdisk -l sudo fdisk /dev/sdb label:gpt，n: 新建分区，p：显示分区 sudo mkfs -t ext4 /dev/sdb1

  • 迁移目录

sudo mkdir /mnt/newpart sudo mount /dev/sdb1 /mnt/newpart cd /home sudo cp -ax * /mnt/newpart mkdir home

fstab

sudo mkdir /home/huhongjun/var cd /var sudo cp -ax * /home/huhongjun/var ln -s /home/huhongjun/var /var

root空间不够迁移docker目录/var/lib/docker

df -h

sudo service docker restart sudo service docker stop

sudo mkdir /home/root-ln cd /var/lib/docker // root-ln下docker目录 sudo cp -r /var/lib/docker /home/root-ln sudo mv /var/lib/docker /var/lib/docker1 sudo ln -s /home/root-ln/docker /var/lib

home目录迁移位其他分区的目录

sudo mkdir /var/home-ln

// root-ln下docker目录 sudo rsync -aXS /home/. /var/home-ln/home/. sudo mv /home /home1 sudo ln -s /var/home-ln/home /

空间清理

sudo apt-get autoclean sudo apt-get clean sudo apt-get autoremove sudo apt-get remove --purge 软件名

  • 删除残余的配置文件 dpkg --list | grep "^rc" dpkg --list | grep "^rc" | cut -d " " -f 3 dpkg --list | grep "^rc" | cut -d " " -f 3 | xargs sudo dpkg --purge sudo dpkg --purge

du -sh /var/cache/apt/archives

  • 清理日志文件 sudo apt-get install ncdu sudo ncdu /var/log sudo dd if=/dev/null of=/var/log/jenkins/jenkins.log

删除大容量软件包 首先安装debian-goodies

sudo apt-get install debian-goodies 然后输入下面的命令

dpigs -H

windows主机名访问

udo apt-get install winbind sudo apt-get install libnss-winbind libpam-winbind sudo vi /etc/nsswitch.conf hosts: =>这一行最后加上wins

sudo apt-get install nfs-common mkdir mnt-ds215j-nfs-ds3617xs sudo mount -t nfs 192.168.1.122:/volume2/NFS-ds3617xs /home/huhongjun/mnt-ds215j-nfs-ds3617xs sudo umount /home/huhongjun/mnt-ds215j-nfs-ds3617xs

Grub默认菜单设置

sudo nano /etc/default/grub
sudo update-grub

ubuntu屏幕字体

Ubuntu 16.04调节屏幕显示字体大小 对于高分辨屏幕来说，Ubuntu的字体可能会有点小，反之，低分率的屏幕字体有点大，设置方法如下： 【设置】->【显示】->【菜单和标题栏缩放比例】

cd /home/huhongjun tar -zxvf /home/huhongjun/mnt-ds215j-nfs-ds3617xs/00-inbox/

虚拟终端中文乱码

ctrl+alt+F1/ctrl+alt+F7 切换终端/图形界面

方案1 - 把系统转成英文的

sudo gedit /etc/default/locale

// 将内容改为： LANG=”en_US.UTF-8″ LANGUAGE=”en_US:en”

sudo locale-gen

方案2 - 安装zhcon软件包

sudo apt-get install zhcon

sudo add-apt-repository ppa:gerardpuig/ppa sudo apt update sudo apt install ubuntu-cleaner

sudo du -sh /var/cache/apt/archives sudo apt-get clean sudo apt-get autoremove --purge BleachBit

cockpit

code /etc/apt/sources.list deb http://us.archive.ubuntu.com/ubuntu/ xenial-backports main restricted universe multiverse sudo apt-get update sudo apt-get -y install cockpit sudo systemctl start cockpit sudo systemctl enable cockpit
