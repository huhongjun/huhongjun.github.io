DS3617xs 使用指南

共享目录 /docker /ARun/docker-volume

docker全部容器的配置文件在这儿 cd /var/packages/Docker/etc/

客户端使用（ds3617xs）

镜像加速与私服设置

ssh admin@192.168.1.172 sudo -i docker images vi /var/packages/Docker/etc/dockerd.json

{
   "data-root" : "/var/packages/Docker/target/docker",
   "log-driver" : "db",
   "registry-mirrors" : ["https://n4tltdbj.mirror.aliyuncs.com","https://registry.docker-cn.com"],
   "insecure-registries": ["192.168.1.172:9005"]
}

curl 192.168.1.172:9005/

重启服务： synoservice --restart pkgctl-Docker curl http://18.163.140.251:2375/images/json curl http://18.163.140.251:2375/images/json

镜像加速

群晖docker使用阿里云镜像加速
vi /var/packages/Docker/etc/dockerd.json
{
        "ipv6": true,
        "registry-mirrors": ["https://?????.mirror.aliyuncs.com"]
}
synoservice --restart pkgctl-Docker

docker api 远程访问

vi /var/packages/Docker/etc/dockerd.json

"hosts": ["unix:///var/run/docker.sock", "tcp://192.168.2.99:2375"]

synoservice --restart pkgctl-Docker curl localhost:2375/images/json netstat -anp|grep 2375 curl 127.0.0.1:2375/info
