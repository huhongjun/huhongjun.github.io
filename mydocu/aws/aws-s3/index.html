aws-s3

官方中文文档

使用现状

存储桶

  • asmatrix.com 空
  • course.asmatrix.com 域名demo.asmatrix.com
  • dev.asmatrix.com mkdocs-asmatrix静态内容，可用BC同步

静态网站部署

AWS部署 1. S3建一个存储桶,存储内容； 2. 设置公开策略（json）； 3. 存储桶开启静态网站托管； 4. Godaddy域名管理中cn指向S3存储桶的域名；

备注

子目录下多个入口的已经有了index.html文件 => 考虑在根目录下做一个html文件
用BC同步文件到S3

aws cli

查看

aws s3 ls
aws s3 ls s3://bucket
aws s3 ls s3://bucket/prefix

拷贝

aws s3 cp /to/local/path s3://bucket/prefix
aws s3 cp s3://bucket/prefix /to/local/path
aws s3 cp s3://bucket1/prefix1 s3://bucket2/prefix2

同步

aws s3 sync --delete /to/local/dir s3://bucket/prefixdir aws s3 sync --delete s3://bucket/prefixdir /to/local/dir aws s3 sync --delete s3://bucket1/prefixdir1 s3://bucket2/prefixdir2 --delete表示文件已存在时先删除再拷贝
