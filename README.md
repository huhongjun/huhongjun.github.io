# 关于本项目

仓库用途：

    - 发布静态网页，作为个人导航门户。
    - 个人github使用记录。

## 日常维护

编辑scripts.js收录网站；

huhongjun/huhongjun.github.io，可以用huhongjun.github.io访问gh-page；
